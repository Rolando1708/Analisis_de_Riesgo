Vamos a simular el comportamiento del precio de un activo que esta determinado por la ecuacin

\[ dS=0.25Sdt+0.15SdW \]

Con parametros anuales con valor inicial de 10. Usaremos este comportamiento para determinar probabilidades sobre el precio, lo primero que necesito es determinar quien es S, que sigue el comportamiento descrito por la ecuacin ds.

Para determinar S, usamos el lema de ITO y obtenemos:

\[S_{t+1}=S_t e^{(0.25-\frac{1}{2}0.15^2)dt+0.15 dw}\]

Usaremos la ecuacin para generar simulaciones y obtener lo siguiente:

Precio Esperado para dentro de un ao Probabilidad de que el precio sea mayor a 10 en un ao Probabilidad de que el precio supere 12 en una ao Probabilidad de que el precio se ubique entre 11 y 13 en un ao Intervalo de confianza para el precio en un ao al 90%

set.seed(123)
So <- 10
mu <- 0.25
sigma <- 0.15
T <- 1 #El tiempo se escribe em aos. utilizamos 1 porque vamos a predecir dentro de un ao
dt <- T/252 #Se divide entre 252 para poder ver la simulacion completa aunque el unico punto que nos importa es el final
t <- seq(0,T,length.out=252)

Vamos a simular dw, el proceso de Wiener

W <- c(0,cumsum(sqrt(dt)*rnorm(251)))
St1 <- So*exp((mu-.5*sigma^2)*t+sigma*W)
plot(t,St1, type="l", col='purple')

Vamos a hacer multiples simulaciones de este comportamiento

sim <- 1000000
p_final <- numeric(sim) # aqui vamos a guardar el precio final, el cual correspondel al precio anual de cada simulacion

for(i in 1:sim){
  W <- c(0,cumsum(sqrt(dt)*rnorm(251)))
  St1 <- So*exp((mu-.5*sigma^2)*t+sigma*W)
  p_final[i] <- St1[252]
}

La distribucion de probabilidad del precio para dentro de un ao con un millon de simuaciones es;
